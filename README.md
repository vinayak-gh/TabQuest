🚀 TabQuest
Eliminate Tab Fatigue. Reclaim Focus.

TabQuest is a Manifest V3 browser extension that serializes your active tabs into "Missions," clearing your workspace for deep work and restoring it only when you're ready.

🔴 The Problem
Too many open tabs = "Context Switching" and high memory usage. It shatters your focus and slows down your computer.

🟢 The Solution
TabQuest "hibernates" your current work. It saves your tabs, closes them to give you a clean slate, and uses a background timer to remind you to resume later.

🛠 How to Use
Capture: Open the extension and name your current session.

Timer: Set how long you want to stay focused.

Launch: Watch your tabs vanish. Focus on your single task.

Resume: Click the system notification to instantly restore all tabs.

💻 Tech Stack
JavaScript: Logic and state management.

Chrome APIs: storage, alarms, notifications.

CSS3: Glassmorphism UI design.

📥 Installation
1.Download this folder.

2.Go to chrome://extensions.

3.Enable Developer Mode.

4.Click Load Unpacked and select this folder.
